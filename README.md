# Bloodborne Enhanced Co-op

This mod currently does the following:
-Removes Multiplayer Fog Gates
-Sets the Beckoning Bell item's insight requirement to 0
-Grants the player all items necessary to begin Co-op
-Removes the ability for the Bell Bearing Woman to appear

This mod is in very early alpha, and is not currently usable at the moment as there are no working methods to play Co-op on emulated versions of Bloodborne at this time.
